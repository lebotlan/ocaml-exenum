Module Exenum_lwt.Lwt_tester

Lwt version of Exenum.tester

val tester : 'a Exenum.t ‑> ?⁠from:Big_int.big_int ‑> ?⁠upto:Big_int.big_int ‑> ?⁠verbose_period:int ‑> ?⁠tos:('a ‑> string) ‑> len:int ‑> ('a ‑> unit Lwt.t) ‑> unit Lwt.t

Note that tester does not comply with non-lwt exceptions: some exceptions may get lost.

  • Parameter verbose_period: A message is printed every verbose_period tests.
  • Parameter tos: Function used to print values of the enumeration.
  • Parameter len: len consecutive values are enumerated, then the index is doubled (and we loop)
  • Parameter upto: By default, the tests go on forever.
