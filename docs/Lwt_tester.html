Module Lwt_tester

module Lwt_tester: sig .. end

Lwt version of Exenum.tester 


val tester : 'a Exenum.t ->
       ?from:Z.t ->
       ?upto:Z.t ->
       ?verbose_period:int ->
       ?tos:('a -> string) -> len:int -> ('a -> unit Lwt.t) -> unit Lwt.t
Note that tester does not comply with non-lwt exceptions: some exceptions may get lost.
upto : By default, the tests go on forever.
verbose_period : A message is printed every verbose_period tests.
tos : Function used to print values of the enumeration.
len : len consecutive values are enumerated, then the index is doubled (and we loop)
